HTML

HTML (de HyperText Markup Language) é uma linguagem de marcação de texto através das TAGs, que permite apresentar informações na internet; como links, imagens e o próprio texto. Todas as páginas que você abre são a leitura e a interpretação de um arquivo que contém esse código HTML.

Elemento básico HTML

html
Representa a raiz de um documento HTML. Todos os outros elementos devem ser descendentes deste elemento.

Metadados do documento

base
Especifica o endereço (URL) utilizado por todos os outros endereços relativos que estão contidos dentro de um documento. Há um número máximo de 1 elemento Base do documento.

head
Providencia informações gerais (metadados) sobre o documento, incluindo seu título e links para scripts e folhas de estilos.

link
Especifica as relações entre o documento atual e um recurso externo. Possíveis usos para este elemento incluem a definição de uma estrutura relacional para navegação. Este elemento é mais usado para vincular as folhas de estilo.

meta
Define qualquer informação de metadados que não podem ser definidas pelos outros elementos HTML, tais como: base, link, script, style ou title.

style
Contém informações de estilo para um documento ou parte dele.

title
Define o título do documento que é apresentado na barra do navegador ou na aba da página.

Separação de conteúdo

address
Apresenta informações de contato para o seu ancestral article ou body mais próximo; em outro caso, ele aplica-se ao documento como um todo.

article
Representa uma composição independente de um documento. Esta tag pode ser um post de um fórum, um artigo de jornal ou revista, um post de um blog, um comentário feito por algum usuário de uma página, aplicação ou um site, um gadget ou widget interativos, ou até mesmo quaisquer formas de conteúdos independentes.

aside
Representa uma seção de uma página que consiste de conteúdo tangencialmente relacionado ao conteúdo do entorno da mesma. Tais seções podem representarem barras laterais. E muitas vezes contém explicações laterais, como a definição de um glosário.

footer
Representa um rodapé para o conteúdo de seção mais próximo ou que tenha parentesco mais próximo de si (tais como: article, aside, nav, section, blockquote, body, details, fieldset, figure, td). Geralmente um rodapé contém informações sobre o autor, direitos autorais ou links para documentos ou páginas relacionadas.

header
Representa um grupo de suporte introdutório ou navegacional. Além disso, pode possuir elementos de cabeçalho, logo, formulário de pesquisa e outros.

h1, h2, h3, h4, h5, h6
Representam seis níveis de título de seção, onde h1 é o nível de seção mais alto e h6 é o mais baixo.

hgroup
Esta tag é destinada para agrupar cabeçalhos de diferentes níveis para uma seção do documento.

main
Define o conteúdo principal da tag body de um documento ou aplicação.

nav
Representa uma seção de uma determinada página que aponta para outras páginas ou até mesmo para outras partes da mesma, ou seja, uma seção com links de navegação.

section
Representa uma seção genérica que está contida no documento HTML, podendo ter um título.

Conteúdo textual

blockquote
Indica que o texto apresentado é uma citação longa.

dd
Esta tag inclui os atributos globais como sendo seus.

dir
É utilizado como um contêiner para um diretório de arquivos, tendo potencial a estilos e ícones aplicados pelo user agent.

div
É um contêiner genérico para conteúdo de fluxo, que de tal forma não Representa algo. Pode ser usado para agrupar elementos para fins de estilos (através de um id ou uma class).

dt
Serve para identificar um termo na lista de definição.

dl
Engloba uma lista de pares de termos e descrições.

hr
Representa uma quebra temática entre elementos de nível de parágrafo como, por exemplo, uma mudança de cena de uma história, ou mesmo uma mudança de tema com uma seção.

li
É utilizado para representar um item que faz parte de uma lista.

ol
Representa uma lista de itens ordenados.

p
Representa um parágrafo do texto.

pre
Esta tag é utilizada para representar um texto pré-formatado. Um texto dentro desse elemento é exibido em uma fonte não proporcional da mesma maneira em que é exibido o texto original do arquivo. Os espaços em branco são devidamente mantidos no texto, de tal forma que este foi digitado.

ul
Representa uma lista de itens desordenados.

Semânticas textuais inline

a
Define uma hiperligação (hyperlink).

abbr
Representa uma abreviação e opicionalmente fornece uma descrição completa para a mesma.

b
Representa um intervalo de texto estilísticamente diferente do texto normal, sem que seja transmitido qualquer importância ou relevância. É geralmente utilizado para destacar palavras-cheves em um resumo e quaisquer textos cuja sua apresentação é em negrito.

bdi
Serve para incorporar o texto com uma direção desconhecida.

bdo
É utilizado para substituir a direção atual de um texto.

br
Representa uma quebra de linha em um texto.

cite
Representa uma referência a uma determinado trabalho artístico. Este elemento deve incluir o título do trabalho ou uma URL como referência.

code
Esta tag define o estilo do texto como sendo um código de computador, sendo devidamente utilizada para frases.

data
Ela vincula um determinado conteúdo a uma tradução legível por máquina. Caso o conteúdo for relacionado à hora ou à data, a tag time deve ser usada.

dfn
Representa uma instância de definição de um dado termo.

em
Ela marca o texto que tem ênfase.

i
Destaca parte do texto do restante em itálico.

kbd
É uma tag de frases; ela define a estilização do texto como sendo uma entrada do teclado.

mark
Representa um trecho de destaque de um texto. Pode ser utilizado em uma página que mostra os resultados de uma busca onde todas as instâncias das palavras pesquisadas receberam destaque.

q
Define citação curta. Normalmente é inserido aspas ao redor da citação por parte dos navegadores.

rb
É utilizado para fins de delimitação de um componente de texto base de uma anotação ruby.

rp
Esta tag pode ser usada para fornecer parênteses em torno de um texto ruby.

rt
Serve para mostrar a pronúncia de caracteres do leste asiático; e posto dentro do elemento ruby.

rtc
Engloba anotações semânticas de caracteres apresentadas em ruby.

ruby
Representa uma anotação ruby (com finalidade de apresentar pronúncias de caracteres do leste asiático).

s
Deixa especificado um texto que não está mais correto, sendo assim, riscando-o.

samp
Define a estilização de saída como sendo de código de computador; esta tag serve para frases.

small
Define o texto com um estilo menor pré-definido, sendo normalmente por padrão com o seguinte atributo: font-size: smaller.

span
É um contêiner genérico em linha de conteúdo fraseado. Pode ser usado para agrupar elementos para fins de estilização (acompanhando os atributos class ou id).

strong
Esta tag define um texto importante, colocando-o em negrito.

sub
Define o texto subscrito.

sup
Define o texto subrescrito.

time
Representa o tempo tanto no formato horas, quanto uma data.

u
Representa um texto apresentado com o uso do tipo sublinhado.

var
Representa uma variável de uma expressão matemática ou em um contexto de programação.

wbr
Representa uma posição no texto onde o navegador pode, opicionalmente, quebrar uma linha.

Imagem e multimídia

area
Define uma região hot-spot em uma imagem. Além disso, só pode ser usado dentro de um elemento map.

audio
Usado para acrescentar conteúdo de som no documento HTML.

img
Usado para acrescentar imagem no documento HTML.

map
É usado com o elemento area para poder definir um mapa de imagem (sendo a área clicável do link).

track
Esta tag permite que seja especificada faixas de texto temporizadas, por exemplo, no uso automático de legendas.

video
Especifica um vídeo, como um clipe de filme ou demais transmissões de vídeo.

figure
Especifica um conteúdo independente, como ilustrações, diagramas, fotos, etc.

figcaption
Define uma legenda para um dado elemento figure.

Conteúdo integrado

applet
Serve para identificar a inclusão de um applet Java.

embed
Esta tag representa um ponto de integração para uma aplicação externa ou conteúdo interativo (chamado de plug-in).

iframe
Representa um contexto de navegação aninhada, efetivamente incorporando outra página HTML para a atual.

object
Define um objeto incorporado em documento HTML.

param
Esta tag é usada a fim de definir parâmetros para plugins incorporados.

picture
É um contêiner utilizado para especificar vários elementos source para um elemento específico img que está contido nele.

source
É utilizado para especificação de vários recursos de mídia, sendo normalmente usado a fim de disponibilizar múltiplos formatos que é suportados por diversos navegadores.

Scripting

canvas
É usado para desenhar gráficos em conjunto com scripts.

noscript
Define uma seção HTML a ser inserida, caso aconteça de um tipo de script não é suportado pela página ou então, caso o script esteja desativado no navegador.

script
Usado para acrescentar ou referenciar um dado script executável.

Demarcando edições

del
Representa uma determinada parte de um texto excluído do documento.

ins
Esta tag representa um intervalo de texto adicionado a um documento.

Conteúdo tabulado

caption
Representa o título de uma dada tabela.

col
É útil para aplicação de estilos em colunas inteiras.

colgroup
Especifica um dado grupo de uma ou mais colunas em uma tabela a fim de formata-la.

table
Define uma tabela.

tbody
Usada a fim de agrupar o conteúdo do corpo de uma dada tabela HTML.

td
Define uma célula padrão de uma dada tabela HTML que contenha em si dados.

tfoot
É usada para agrupamento de conteúdo do rodapé em uma dada tabela HTML.

th
Define uma célula de cabeçalho em uma dada tabela HTML.

thead
É usada a fim de agrupar o conteúdo do cabeçalho em uma tabela HTML.

tr
Define uma linha em uma dada tabela HTML. Pode conter um ou mais elementos th ou td.

Formúlarios

button
Representa um botão clicável.

datalist
Esta tag contém diversos elementos option que são as opções possíveis para o um valor de dados outros controles.

form
Representa uma seção de um documento que possui controles interativos e que perrmitem ao usuário fornecer informação a um servidor web.

input
Esta tag especifica um campo de entrada, onde o usuário pode inserir dados.

option
Serve para criar um controle que representa um item dentro de um elemento, tais como, select ou datalist em um formulário.

output
É usado para que um website ou aplicação possa injetar os dados resultantes de um cálculo ou um dado resultado de uma ação do usuário.

progress
Este elemento serve para visualização de um dado progresso de uma determinada tarefa.

select
Representa um controle que apresenta um menu de opções. Essas opções podem ser pré-selecionadas para o usuário.

Elementos interativos

details
É usado como uma ferramenta onde o usuário possa obter informações adicionais.

dialog
Representa uma caixa de diálogo ou outro componente interativo, como um inspetor ou janela.

CSS

CSS (Cascading Style Sheets ou Folhas de Estilo em Cascata) é uma linguagem de estilo (en-US) usada para descrever a apresentação de um documento escrito em HTML ou em XML (incluindo várias linguagens em XML como SVG, MathML ou XHTML). O CSS descreve como elementos são mostrados na tela, no papel, na fala ou em outras mídias.

Seletores CSS

.class
Seleciona todos os elementos com class="elementos", por exemplo.

#id
Seleciona o elemento com id="elemento", por exemplo.

*
Seleciona todos os elementos.

elemento
Seleciona todos os elementos, tal como um parágrafo (< p >), por exemplo.

elemento, elemento
Seleciona todos os elementos de uma tag e também de outra tag ao mesmo tempo.

elemento elemento
Seleciona todos os elementos de uma tag dentro de elementos de outra tag.

elemento > elemento
Seleciona todos os elementos filho de um elemento pai na árvore DOM HTML.

elemento + elemento
Seleciona todos < p > que estão imediatamente depois de um elemento < div >.

elemento1 ~ elemento2
Seleciona todos os elementos de um lista não ordenada < ul > que precede um elemento < p >, por exemplo.

[attribute]
Seleciona todos os elementos com um dado atributo.

[attribute = value]
Seleciona todos os elementos com o dado valor atribuído ao dado atributo.

[attribute ~= value]
Seleciona todos os elementos com um dado atributo que contém um valor chave como, por exemplo, [title ~= valor].

[attribute |= value]
Seleciona todos os elementos com um dado atributo, cujo tenha um dado valor atribuído a ele como, por exemplo, [lang |= pt-BR].

[attribute ^= value]
Seleciona todo um elemento que tem um dado atributo, cujo valor sirva como chave como, por exemplo, o elemento < a > tem um atributo href e seu valor começa com "https"; ficando deste modo na sintaxe do CSS: a[href ^= "https"].

:active
Seleciona o link ativo como, por exemplo, a:active.

::after
Insere conteúdo após um dado elemento como, por exemplo, p::after.

::before
Insere conteúdo antes de um dado elemento como, por exemplo, p::before.

:checked
Seleciona todo um dado elemento que está marcado com checked.

:first-child
Seleciona todo um dado elemento que é o primeiro filho do seu pai.

::first-letter
Seleciona o primeiro dado de todo um elemento.

::first-line
Seleciona a primeira linha de todo um dado elemento.

JavaScript

JavaScript (às vezes abreviado para JS) é uma linguagem leve, interpretada e baseada em objetos com funções de primeira classe, mais conhecida como a linguagem de script para páginas Web, mas usada também em vários outros ambientes sem browser, tais como node.js, Apache CouchDB e Adobe Acrobat. O JavaScript é uma linguagem baseada em protótipos, multi-paradigma e dinâmica, suportando estilos de orientação a objetos, imperativos e declarativos (como por exemplo a programação funcional).

Instruções e Declarações

break (Controle de Fluxo)
Encerra o loop atual, troca ou nomeia instruções e transfere o controle do programa para a próxima instrução.

continue (Controle de Fluxo)
Termina a atual execução de instruções do loop e continua a executar o loop com a seguinte iteração.

if ... else (Controle de Fluxo)
Executa uma instrução se a dada condição é verdadeira. Caso contrário, ou seja, a dada condição seja falsa, outra instrução pode ser executada.

switch (Controle de Fluxo)
Retorna uma instrução cujo o valor da expressão seja combinado a condição do caso, sendo assim, o dado bloco é executado sendo associado ao dado caso.

throw (Controle de Fluxo)
Aplica uma execução que é definida pelo usuário.

try ... catch (Controle de Fluxo)
Marca um bloco de instruções para especificação de uma resposta e, tentar, um retorno de uma execução.

Bloco (Controle de Fluxo)
Uma instrução de bloqueio que é utilizada para um dado grupo de zero ou mais instruções. Este bloqueio é delimitado por um par de chaves ( {...} ).

Vazio (Controle de Fluxo)
Uma instrução vazia é utilizada para dar uma "não-instrução", mesmo que a linguagem de programação JavaScript espere por uma (ou algo acontecer).

var (Declaração)
Declara uma variável, podendo ser opicional a inicialização com um dado valor à ela.

let (Declaração)
Declara acesso ao local da variável, podendo ser opcional a inicialização com um dado valor.

const (Declaração)
Declara uma constante, sendo apenas para leitura.

function (Funções e Classes)
Declara uma função com os parâmetros especificos.

function* (Funções e Classes)
Cria funções de modo que habilita iteradores de escrita com mais facilidade.

return (Funções e Classes)
Deixa especificado o valor a ser retornado por uma dada função.

class (Funções e Classes)
Declara uma classe.

do ... while (Iteração)
Cria uma estrutura de repetição que executa uma dada instrução até que a condição de teste seja falsa.

for (Iteração)
Cria uma estrutura de repetição que considera três expressões entre parenteses e sepados por ponto e vírgula, seguido pela instrução executada. Por exemplo, for (var index = 0; index < algumValor; i = i + 1) { ... }.

for ... in (Iteração)
Itera através de enumeráveis propriedades de um objeto, em ordem arbitária. Assim, para cada propriedade distinta, instruções podem ser executadas.

for ... of (Iteração)
Itera através de objetos iteráveis (como arrays, array-like objects, iterators & generators), assim, invocando uma iteração personalizada com instruções para serem executadas pelo valor de cada propriedade.

while (Iteração)
Cria uma estrutura de repetição que executa uma especifica instrução enquanto o teste da condição for verdadeiro.

debugger
Invoca qualquer funcionalidade de depuração à disposição. Caso não haja nehuma função disponível, a instrução não terá efeito algum.

export
É utilizado para exportar funções.

import
É utilizado para importar funções que são exportadas.

label
Serve para fornecer uma instrução com um identificador que pode ser referido utilizando as instruções break ou continue.

Operadores do JavaScript

+ (Operador Aritmético)
Adição

- (Operador Aritmético)
Subtração

* (Operador Aritmético)
Multiplicação

** (Operador Aritmético)
Exponenciação

/ (Operador Aritmético)
Divisão

% (Operador Aritmético)
Módulo

++ (Operador Aritmético)
Incremento

-- (Operador Aritmético)
Decremento

= (Operador de Atribuição)
Exemplo: x = y; é equivalente à: x = y.

+= (Operador de Atribuição)
Exemplo: x += y; é equivalente à: x = x + y.

-+ (Operador de Atribuição)
Exemplo: x -= y; é equivalente à: x = x - y.

*= (Operador de Atribuição)
Exemplo: x *= y; é equivalente à: x = x * y.

/= (Operador de Atribuição)
Exemplo: x /= y; é equivalente à: x = x / y.

%= (Operador de Atribuição)
Exemplo: x %= y; é equivalente à: x = x % y.

<<= (Operador de Atribuição)
Exemplo: x <<= y; é equivalente à: x = x << y.

>>= (Operador de Atribuição)
Exemplo: x >>= y; é equivalente à: x = x >> y.

<<<= (Operador de Atribuição)
Exemplo: x <<<= y; é equivalente à: x = x <<< y.

&= (Operador de Atribuição)
Exemplo: x &= y; é equivalente à: x = x & y.

^= (Operador de Atribuição)
Exemplo: x ^= y; é equivalente à: x = x ^ y.

|= (Operador de Atribuição)
Exemplo: x |= y; é equivalente à: x = x | y.

**= (Operador de Atribuição)
Exemplo: x **= y; é equivalente à: x = x ** y.

& -> AND (Operador Binário)
Define cada bit (dado) como 1 (verdadeiro) se os demais bits (condições) forem 1 (verdadeiros).

| -> OR (Operador Binário)
Difine cada bit (dado) como 1 (verdadeiro) se algum bit (condição) for 1 (verdadeiro).

^ -> XOR (Operador Binário)
Define cada bit (dado) como 1 (verdadeiro) se apenas um dos demais bits (condições) for 1 (verdadeiro).

~ -> NOT (Operador Binário)
Inverte todos os bits (dados).

== (Operador de Comparação)
Igual a; por exemplo, x == y; é equivalente à: x é igual a y.

=== (Operador de Comparação)
Valor igual e, também, tipo igual; por exemplo, x === y; é equivalente à: x = 1 e y = 1, sendo ambos do tipo inteiro.

!= (Operador de Comparação)
Diferente de; por exemplo, x != y; é equivalente à: x é diferente de y, pois, x = 1 e y = 2.

!== (Operador de Comparação)
Valor diferente e, também, tipo diferente; por exemplo, x !== y; é equivalente à: x = "1" e y = 2, sendo que x é do tipo caractere e y é do tipo inteiro.

> (Operador de Comparação)
Maior que...

< (Operador de Comparação)
Menor que...

>= (Operador de Comparação)
Maior ou igual a...

<= (Operador de Comparação)
Menor ou igual a...

? : (Operador de Comparação)
Operador ternário. Funciona da seguinte forma: valor = (condição) ? x : y; sendo assim, caso a dada condição seja verdadeira, o valor será x, caso contrário, será y.

|| (Operador Lógico)
OU (OR).

&& (Operador Lógico)
E (AND).

! (Operador Lógico)
Negação (NOT).
